StoreIt ☁️📦

StoreIt is a modern cloud storage web application that allows users to upload, store, organize, and manage their files securely from anywhere.

The app provides a clean interface and fast file management experience with authentication and cloud storage integration.

🚀 Features

🔐 Secure user authentication

📤 Upload files to cloud storage

📁 Organize files easily

📄 Preview uploaded files

🗑️ Delete files anytime

📊 Track file storage usage

📱 Fully responsive design

🛠️ Tech Stack

This project uses modern technologies:

Frontend: Next.js

Language: TypeScript

Styling: Tailwind CSS

Backend / Storage: Appwrite

Version Control: Git

📦 Installation

1. Clone the repository
   git clone https://github.com/huzeifaziyad-spec/storage_management_solution-main.git
2. Navigate into the project folder
   cd storage_management_solution-main
3. Install dependencies
   npm install
4. Setup environment variables

Create a .env.local file and add:

NEXT_PUBLIC_APPWRITE_ENDPOINT=
NEXT_PUBLIC_APPWRITE_PROJECT=
NEXT_PUBLIC_APPWRITE_DATABASE=
NEXT_PUBLIC_APPWRITE_USERS_COLLECTION=
NEXT_PUBLIC_APPWRITE_FILES_COLLECTION=
NEXT_PUBLIC_APPWRITE_BUCKET=
NEXT_APPWRITE_KEY= 5. Run the development server
npm run dev

Open the app in your browser:

http://localhost:3000
